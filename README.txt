etalonPO.csv - Ваша база эталонного ПО, формируете сами одним столбцом Names

ps_equal - сам скрипт

В папку с проектом копируются *.cvs файлы с заголовком таблицы H1 с програмным обеспечением рабочих станций.

После запуска скрипта формируются файлы с начинающимся именем 'Complite', в которых остается все ПО, которое не входит в etalonPO.csv

Базу эталонного ПО нужно формировать исходя из разных вариаций названий ПО, для более гибкого поиска

Пример:
7-zip
7zip
